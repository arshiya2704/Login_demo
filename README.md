# Login_demo

Login page using technologies:

• Frontend: ejs </br>
• Backend:  node.js
• Database: MySQL

Features: 

• Login</br>
• Register

Steps to run the application:
1. Import the project in Eclipse
2. Right click and Run as -> npm install
3. Right click and Run as -> Node.js Application
